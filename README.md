# Interview Buddy v2

Interview Buddy v2 is a web-based interview preparation platform designed to help users practice, improve, and gain confidence for technical and HR interviews. The project focuses on providing a smooth, interactive experience that simulates real interview scenarios while keeping the interface simple and accessible.

Live Demo:  
https://interview-buddy-v2.vercel.app

---

## Features

- Interactive interview-style questions  
- Clean and responsive user interface  
- Fast performance with modern web technologies  
- Designed to simulate real interview flow  
- Easy to use and beginner-friendly  

---

## Tech Stack

- Frontend Framework: React / Next.js  
- Styling: CSS / Tailwind CSS  
- Deployment: Vercel  

---

## Getting Started

Follow these steps to run the project locally.

### Prerequisites
- Node.js installed on your system  
- npm or yarn package manager  

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/interview-buddy-v2.git
   ```

2. Navigate to the project directory
   ```bash
   cd interview-buddy-v2
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm run dev
   ```

5. Open your browser and visit
   ```
   http://localhost:3000
   ```

## Project Purpose

The goal of Interview Buddy v2 is to help students and job seekers practice interview questions in a structured and realistic way. It is especially useful for those preparing for technical roles or placement interviews.
